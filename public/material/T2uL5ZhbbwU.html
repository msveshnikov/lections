Следователь, приступая к производству предъявления для опознания, разъясняет участвующим их права и обязанности. Затем опознаваемому предлагается занять место по его выбору среди других предъявляемых граждан. После этого приглашается опознающий, которому предлагается внимательно осмотреть предъявленных в тех позах, в которых они находятся. Положение предъявляемых может быть изменено по желанию опознающего или по усмотрению следователя. Торопить опознающего нельзя.

В случае опознания определенного лица опознающим должны быть названы признаки, по которым он его опознал; при отрицательном результате опознания опознающий должен объяснить причины этого.

Если возникает необходимость предъявления нескольких лиц одному опознающему, то предъявление для опознания каждого из них производится порознь.

Предъявление для опознания человека может быть произведено по динамическим (функциональным) признакам: по особенностям голоса, речи и походки:

При предъявлении для опознания по голосу и особенностям речи голосовая характеристика и речь предъявляемых не должны иметь существенных различий. Признаками устной речи, используемыми при опознании, служат дефекты и специфические обороты речи и слова. Признаки голоса — это его высота, сила и тембр.

Опознаваемый сам избирает очередность прослушивания его голоса и речи. Затем в присутствии понятых следователь ведет заранее продуманную беседу с предъявляемыми, стремясь, чтобы их речь звучала достаточно продолжительное время. В зависимости от обстоятельств дела следователь может предложить предъявляемым повысить или понизить голос, повторить какую-то фразу, слово, букву. Целесообразно при этом вести магнитофонную запись.

При предъявлении для опознания по походке предъявляемые должны иметь сходство в походке. Следует заранее условиться о сигналах, которые следователь будет подавать предъявляемым в целях изменения направления, скорости движения, поз и о номерах, под которыми они будут фигурировать. Называть предъявляемых по фамилиям нельзя. Затем следователь предлагает опознаваемому определить, каким по счету он желает идти, и намечает маршрут. В случае опознания выясняется, по каким приметам походки опознан человек. Процесс ходьбы желательно зафиксировать с помощью видеозаписи.
